# Info
Requires NUMLOCK to be ENABLED:
- NUMPAD4: Previous Song
- NUMPAD5: Pause/Play
- NUMPAD6: Skip song

Spotify only:
- Numpad8: Increase Volume by 0.05
- Numpad2: Decrease Volume by 0.05
- Displays current song playing
